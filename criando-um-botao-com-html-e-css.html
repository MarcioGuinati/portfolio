Criando botão html e css

Você vai aprender agora a criar seu primeiro botão CSS, primeiramente crie uma pasta com o nome do projeto conforme mostrado abaixo

Feito o passo acima, agora crie o seu conteudo com html, pode dar o nome que quiser, mas com a extensão .html, exemplo index.html, no caso o print abaixo eu criei como botaoSalvar.html

Legal!! Agora vamos colocar a mão na massa, criado o seu arquivo com extensão .html, vamos programar hehe

                div
                    button type="submit" name="salvar" id="salvar" onclick="salvar()">Salvar button
                div

Show de bola, codigo html feito, agora vamos estilizar esse botão acima!

                body{
                    background-color: blueviolet;
                    margin: 0px;
                    padding: 0px;
                }
                #salvar{
                    background-color: #00BFFF;
                    color: #FFFFFF;
                    font-family: Arial, Helvetica, sans-serif;
                    width: 150px;
                    height: 40px;
                    border: solid 1px #000000;
                    cursor: pointer;
                    transition-duration: 2s;
                }
                #salvar:hover{
                    background-color: #1E90FF;
                    color: #FFFFFF;
                }
                div{
                    text-align: center;
                    padding: 250px;
                }
